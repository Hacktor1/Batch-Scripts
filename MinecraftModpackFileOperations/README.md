# Minecraft Mod Manager Folder

This folder contains tools and scripts for managing Minecraft mods efficiently. Its purpose is to simplify tasks like adding, removing, or organizing mods across multiple Minecraft versions.

## Purpose

* Organize your Minecraft mods in one place.
* Automate repetitive tasks such as copying or deleting mods.
* Support multiple Minecraft versions without manual file management.
* Make mod management faster, safer, and less error-prone.

## How to Use

Place your mod files in this folder alongside the scripts, then run the scripts as needed to apply changes to your Minecraft installations. The scripts are designed to work automatically with your installed versions.

## Benefits

* Keeps your Minecraft installation organized.
* Reduces mistakes when updating or removing mods.
* Works for any number of mods or Minecraft versions.

## Notes

* Ensure that your Minecraft folder is in the default `%APPDATA%\.minecraft` location.
* Always back up your mods before performing batch operations.
